# Isaacson Nigeria Limited - Website

## Project Structure
```
isaacson-nigeria-limited/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── images/
│   ├── logo.png              ← Your company logo (white background)
│   ├── hero-import-export.jpg ← Import/export containers image
│   ├── hero-containers.jpg    ← Stacked shipping containers
│   ├── hero-logistics.jpg     ← Air, sea, truck logistics composite
│   └── about-team.jpg         ← Team or office image for About section
└── README.md
```

## Required Images

Please save your images to the `images/` folder with these exact filenames:

| Filename | Description | Notes |
|----------|-------------|-------|
| `logo.png` | Company logo | White background, use the attached logo |
| `hero-import-export.jpg` | Import/Export containers | Hero slider background |
| `hero-containers.jpg` | Stacked containers at port | Hero slider background |
| `hero-logistics.jpg` | Airplane, ship, truck | Hero slider background |
| `about-team.jpg` | Team/office photo | For About section (use any shipping image) |

## How to Add Images

1. Copy your images to: `/Users/mac/isaacson_ltd/isaacson-nigeria-limited/images/`
2. Rename them to match the filenames above
3. Open `index.html` in your browser to view the website
